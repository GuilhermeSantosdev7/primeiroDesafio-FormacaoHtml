Explicação de cada tag aprendida

Tag A

A Tag a é conhecida como Tag ancora, sua utilização cria links para outros sites ou para uma parte especifica da mesma página.


Tag P

A Tag P é utilizada para demarcar um paragrafo dentro do site.


Tag Strong

A Tag strong é utilizada para deixar um texto em negrito, também é utilizada para dar importancia a uma parte do texto.


Tags H

As Tags H1, H2, H3, H4, H5 e H6 são utilizadas para titulos, o tamanho do formato do texto varia de acordo com a tag H escolhida, quando menor o número da tag, maior o titulo.


Tag title

A Tag title é utilizada para informar ao HTML qual o titulo da página.


Tags U e O

A tag U e O server para criação de listas desordenadas e ordenadas respectivamente.


Tag li.

A tag li representa os itens dentro da lista.


Tag input.

A tag input é utilizada para colher dados do usuário, sendo numericos, e-mails ou texto.


Tag img

A tag img é utilizada para inserir imagens dentro do site.

Atributos

Além das tags é possivel utilizar atributos nas tags, dessa forma o código adquiri atributos que podem mudar o comportamento da tag como type, href, class ou src.
